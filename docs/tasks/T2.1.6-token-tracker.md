## 技术方案

### 数据结构

```python
@dataclass
class TokenUsage:
    id: int | None
    provider: str
    model: str
    tokens_in: int
    tokens_out: int
    cost_usd: float
    timestamp: str
    agent_id: str | None = None
    session_id: str | None = None
    metadata: dict[str, Any] = field(default_factory=dict)
```

### TokenTracker API

```python
class TokenTracker:
    def record(self, response: LLMResponse, *, 
               agent_id: str | None = None,
               session_id: str | None = None) -> TokenUsage
    
    def query(
        self,
        *,
        agent_id: str | None = None,
        session_id: str | None = None,
        provider: str | None = None,
        since: datetime | None = None,
        until: datetime | None = None,
        limit: int = 100,
    ) -> list[TokenUsage]
    
    def summary(
        self,
        *,
        agent_id: str | None = None,
        session_id: str | None = None,
        group_by: str | None = None,  # "provider" | "model" | "agent_id"
    ) -> dict[str, Any]
    
    def reset(self) -> None
```

### SQLite Schema

```sql
CREATE TABLE IF NOT EXISTS token_usage (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    provider TEXT NOT NULL,
    model TEXT NOT NULL,
    tokens_in INTEGER NOT NULL,
    tokens_out INTEGER NOT NULL,
    cost_usd REAL NOT NULL,
    timestamp TEXT NOT NULL,
    agent_id TEXT,
    session_id TEXT,
    metadata TEXT  -- JSON
);

CREATE INDEX IF NOT EXISTS idx_agent_id ON token_usage(agent_id);
CREATE INDEX IF NOT EXISTS idx_session_id ON token_usage(session_id);
CREATE INDEX IF NOT EXISTS idx_timestamp ON token_usage(timestamp);
```

### 成本计算

如果 `LLMResponse.cost_usd` 为 None，根据模型价格表估算：

```python
MODEL_PRICING = {
    "deepseek-chat": {"input": 0.14, "output": 0.28},  # per 1M tokens
    "gpt-4": {"input": 30.0, "output": 60.0},
    "gpt-3.5-turbo": {"input": 0.5, "output": 1.5},
    # ...
}
```

## 集成点

### 1. llm.chat() 自动记录

```python
async def chat(...) -> LLMResponse:
    response = await client.chat(...)
    tracker = get_tracker()
    tracker.record(response, agent_id=kwargs.get("agent_id"))
    return response
```

### 2. GenerationPipeline 传入 session_id

```python
class GenerationPipeline:
    async def generate_and_deploy(self, requirement: str, ...) -> GenerationOutcome:
        session_id = f"gen_{uuid4().hex[:8]}"
        # 传给 llm.chat()
        response = await self.llm_chat(messages, session_id=session_id)
```
