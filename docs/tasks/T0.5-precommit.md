## 目标

配置 pre-commit hooks，强制 ruff / mypy / 格式检查在 commit 前跑。

---

## 执行步骤

### Step 1: 创建 `.pre-commit-config.yaml`

```bash
cd /Users/popmart/Documents/python/forge-agent

cat > .pre-commit-config.yaml <<'EOF'
repos:
  - repo: https://github.com/pre-commit/pre-commit-hooks
    rev: v4.6.0
    hooks:
      - id: trailing-whitespace
      - id: end-of-file-fixer
      - id: check-yaml
      - id: check-toml
      - id: check-added-large-files
        args: ['--maxkb=1000']
      - id: check-merge-conflict

  - repo: https://github.com/astral-sh/ruff-pre-commit
    rev: v0.4.9
    hooks:
      - id: ruff
        args: [--fix, --exit-non-zero-on-fix]
      - id: ruff-format

  - repo: https://github.com/pre-commit/mirrors-mypy
    rev: v1.8.0
    hooks:
      - id: mypy
        additional_dependencies: [pydantic>=2.5]
        args: [--config-file=pyproject.toml]
EOF
```

### Step 2: 创建 `.editorconfig`

```bash
cat > .editorconfig <<'EOF'
root = true

[*]
indent_style = space
indent_size = 4
end_of_line = lf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true

[*.{yml,yaml,toml}]
indent_size = 2

[*.md]
trim_trailing_whitespace = false
EOF
```

### Step 3: 安装并运行

```bash
pip install pre-commit
pre-commit install
pre-commit run --all-files

git add .pre-commit-config.yaml .editorconfig
git commit -m "chore: add pre-commit configuration"
```

## 自动化验证命令

```bash
if [ -f .pre-commit-config.yaml ] && [ -f .editorconfig ]; then
    if pre-commit run --all-files > /dev/null 2>&1; then
        echo "[T0.5] pre-commit 配置             ✅ PASS"
    else
        echo "[T0.5] pre-commit 配置             🟡 CONFIG EXISTS, run failed (check output)"
    fi
else
    echo "[T0.5] pre-commit 配置             ❌ FAIL"
    exit 1
fi
```
