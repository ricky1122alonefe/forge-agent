## 目标

实现 **智能重试机制**，当生成/验证/沙箱失败时：

1. **自动重试** — 最多 3 次（可配置），每次用不同策略
2. **失败回滚** — 如果所有重试都失败，回滚到上一个稳定版本
3. **错误分类** — 区分可重试错误（网络超时、LLM 限流）和不可重试错误（语法错误、权限问题）
4. **重试日志** — 记录每次重试的原因、尝试次数、最终结果

from dataclasses import dataclass
from enum import Enum
from typing import Any, Callable, Awaitable

class RetryableError(str, Enum):
    """Errors that can be retried."""
    NETWORK_TIMEOUT = "network_timeout"
    RATE_LIMIT = "rate_limit"
    LLM_OVERLOAD = "llm_overload"
    VALIDATION_FAILED = "validation_failed"
    SANDBOX_TIMEOUT = "sandbox_timeout"

class NonRetryableError(str, Enum):
    """Errors that should NOT be retried."""
    SYNTAX_ERROR = "syntax_error"
    PERMISSION_DENIED = "permission_denied"
    INVALID_CONFIG = "invalid_config"
    FORBIDDEN_IMPORT = "forbidden_import"

@dataclass
class RetryConfig:
    max_attempts: int = 3
    backoff_factor: float = 1.5  # exponential backoff
    retryable_errors: set[RetryableError] = field(default_factory=lambda: {
        RetryableError.NETWORK_TIMEOUT,
        RetryableError.RATE_LIMIT,
        RetryableError.LLM_OVERLOAD,
        RetryableError.VALIDATION_FAILED,
        RetryableError.SANDBOX_TIMEOUT,
    })

class RetryManager:
    def __init__(self, config: RetryConfig | None = None):
        self.config = config or RetryConfig()

def classify_error(error: Exception | str) -> RetryableError | NonRetryableError:
    """Classify an error as retryable or not."""
    error_str = str(error).lower()
    
    if "timeout" in error_str or "timed out" in error_str:
        return RetryableError.NETWORK_TIMEOUT
    if "rate limit" in error_str or "429" in error_str:
        return RetryableError.RATE_LIMIT
    if "overloaded" in error_str or "503" in error_str:
        return RetryableError.LLM_OVERLOAD
    if "syntax error" in error_str:

### Step 4: 回滚机制

```python
class RollbackManager:
    def __init__(self, code_store: FileCodeStore):
        self.code_store = code_store
    
    def rollback(self, agent_id: str) -> SavedCode | None:
        """Rollback to the previous stable version."""
        manifest = self.code_store.manifest
        entry = manifest.agents.get(agent_id)
        if not entry or len(entry.versions) < 2:
            return None
        
        # Find the last stable version (validation_status == "passed")
        for version in reversed(entry.versions[:-1]):
            if version.validation_status == "passed":
                return self.code_store.load(agent_id, version.version)
        return None
```
